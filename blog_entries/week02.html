Week 02 - CSS

After doing well with HTML the previous week. I decided to work on CSS and Responsive pages this week. Before starting the experiments, I read about CSS in w3schools and then I first experimented on "try it yourself". Later I read professor Jose's slides and I started my experiments.

My first experiment is an inline css where the css was applied to each tag. This is not considered as a good way of designing so in my second experiment I moved all the css to <head> part of the html with <style> tag. This is also not considered a good way of designing so I created a separate css file in my third experiment and linked to the html using <link> tag.

Further, I continued working on responsive images in my fourth experiment and responsive navigation bar in fifth experiment where the orientation of the navigation bar changes as the screen size is reduces to a specific size. This is done using @media screen which determines the screen size.

I feel confident on CSS now, Next week I would like to work on Bootstrap.
